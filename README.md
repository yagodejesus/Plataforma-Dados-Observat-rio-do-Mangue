# Plataforma Inteligente para Governança de Dados da Sociobiodiversidade

Este repositório apresenta a arquitetura, documentação técnica e fundamentos científicos de uma plataforma baseada em Inteligência Artificial desenvolvida no âmbito do projeto:

**Governança de uso dos serviços ecossistêmicos através do SiBp-Maretório**

## Objetivo

Desenvolver uma plataforma modular baseada em IA generativa para:

- FAQ inteligente baseado em RAG
- Suporte técnico automatizado
- Sistema de recomendação contextual
- Integração com bases de sociobiodiversidade

## Arquitetura

A plataforma segue arquitetura modular composta por:

- Backend API
- Banco de dados relacional
- Banco vetorial
- Pipeline RAG
- Sistema de classificação de intenção
- Sistema de recomendação híbrido

Mais detalhes em `/docs`.

## Tecnologias

- Python
- FastAPI / Node.js
- PostgreSQL
- FAISS ou ChromaDB
- Modelos de linguagem (LLM)
- Embeddings semânticos

## Contribuição Científica

O sistema integra técnicas de:

- Retrieval-Augmented Generation (RAG)
- Classificação de intenção
- Gestão de contexto conversacional
- Sistemas de recomendação baseados em similaridade vetorial

Artigo científico disponível em `/docs/artigo-cientifico.md`.
